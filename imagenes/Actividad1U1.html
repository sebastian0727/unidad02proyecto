BOOTCAMP DESARR WEB FULL VIRT (PAOLA CUADROS_15022021_C12_202104_I)

Actividad 1 - Editor de código y elementos básicos HTML5

Fecha límite domingo, 28 de febrero de 2021, 23:59
Tiempo restante: 6 días 13 horas
Competencia específica

Identificar los elementos de la interfaz del software para el desarrollo web para estar en la capacidad de  utilizar HTML, CSS y JavaScript de manera autónoma.

Técnica y/o herramienta didáctica

Tarea

Guía de la actividad

Apreciados estudiantes, cordial saludo y una calurosa bienvenida a este primer desafío académico.

Esperamos que este espacio sea la oportunidad para que puedan interiorizar los conceptos claves en este curso. Los invitamos a aprovechar todos los recursos y herramientas tecnológicas que el módulo posee para poder enriquecer su proceso de aprendizaje.

Así mismo, les deseamos lo mejor en esta  experiencia en línea y los animamos a comunicar todas sus inquietudes por los diferentes canales que BIT ha destinado para tal fin.

De manera individual deberán elaborar una página web sobre su artista musical favorito. Por favor, revisen las siguientes instrucciones para que la actividad sea clara y la puedan completar con éxito

1. Realizar la revisión del siguiente texto

    • Celaya, Luna, A. (2014). Creación de páginas web: HTML 5. Recuperado de la base de datos E libro

2. Ver las siguientes videoclases donde está explicado de manera práctica, cómo aplicar cada elemento necesario para el desarrollo de esta actividad tanto en el HTML como en el CSS:



3. Revisar las siguientes presentaciones: 

    • Unidad1_Clase1_EditorDeTexto

    • Unidad1_Clase2_HTML 

    • Unidad1_Clase3_HTML

    • Unidad1_Clase4_HTML 

    • Unidad1_Clase5_HTML

    • Unidad1_Clase6_HTML

    • Unidad1_Clase7_HTML

    • Unidad1_Clase8_HTML

4. Descargar la aplicación Visual Studio Code, haciendo clic en el siguiente enlace: 

https://code.visualstudio.com/

5. Desarrollar una página web utilizando el editor de código Visual Studio Code, tal como se explica en el material de consulta presentado en el punto 1 y 2.  

6. Crear en sus PC o Mac o Linux una carpeta llamada Actividad1U1, ya que en esta carpeta estarán los archivos que se entregarán para esta actividad.

7. Crear un archivo index.html en Visual Studio code dentro de la carpeta anteriormente creada (Actividad1U1), en esta página se va a maquetar el proyecto, se debe poner una imagen utilizando la etiqueta <img>, un título con la etiqueta <h1>, un párrafo con la etiqueta <p>, y un video con la etiqueta <video>.

8. También se debe crear un archivo estilo.css , y debe crearse una etiqueta <link> en el archivo index.html para enlazar este archivo estilo.css donde se configura el diseño para las etiquetas, se debe dar un estilo en el que los elementos imagen, título, párrafo, video estén centrados.

9. Enviar a través de la plataforma un archivo .ZIP en el que se encuentren el index.html y estilo.css con las etiquetas solicitadas. 

10. Revisar la retroalimentación que se generará al finalizar la revisión del ejercicio por parte del tutor de su curso.

Tengan presente:

Si tienen dudas e inquietudes, pueden compartirlas a través del Foro de acompañamiento permanente. En recursos complementarios o de apoyo, pueden encontrar documentos que les servirán para ampliar la información acerca de los temas de la actividad.

 ¡Éxitos en el desarrollo de la actividad!

Entregable

Antes de enviar su página web, verifiquen que esta cumple con las siguientes características:

  • La carpeta llamada Actividad1U1 debe contener los siguientes elementos: 

    • index.html (creado por el estudiante).

    • estilo.css (creado por el estudiante).

    • imagen1 (descargada de internet por el estudiante).

    • video1 (descargado de internet por el estudiante).

  • El archivo index.html se debe tener los siguientes parámetros:

    • Una imagen llamada Imagen1 que haga referencia a un artista musical.

    • Un título con el nombre completo del artista musical seleccionado para la imagen.

    • Un párrafo con una pequeña biografía del artista musical, seleccionado en la imagen.

    • Un video de máximo dos minutos que haga referencia al artista musical seleccionado.

  • En el archivo estilo.css, se debe crear:

    • Un color de fondo gris oscuro para la página. 

    • Un color de fondo blanco para el título.

    • Un color de fondo blanco para el párrafo.

    • Todos los elementos de la página deben ir centrados.

    • Adicionalmente debe contar con muy buena ortografía y sintaxis de código.

Criterios de participación

Los criterios de evaluación  son los que se han relacionado en la rúbrica: 

  • Ortografía y sintaxis. 

  • Contenido de la página. 

  • Apropiación de conceptos. 

  • Multimedia. 

  • Contenido de la carpeta.
